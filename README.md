# 🎛️ Beat Pro — Lyrics to Beat Generator

**Beat Pro** is a modern web application that transforms your lyrics into a musical foundation. By analyzing the sentiment and content of your lyrics, Beat Pro recommends a curated set of instruments, detects the mood, and generates an export-ready beat using its built-in step sequencer.

![Beat Pro Interface](https://raw.githubusercontent.com/z9mb83/beat-pro/main/screenshot.png) *(Note: Placeholder for actual screenshot)*

## ✨ Features

- **Lyric Analysis:** Paste your lyrics and get instant recommendations for mood and instruments.
- **Genre-Specific Workflows:** Choose from standard genres like **Hip-Hop**, **Pop**, and **Rock**, or explore fusion styles like **Indian Classical**, **Goa Trance**, and **Middle Eastern**.
- **Interactive Step Sequencer:** Fine-tune your beat rhythm directly in the browser.
- **Advanced Export Options:**
    - **LMMS Export:** Direct export to `.mmpz` format for use in the Linux MultiMedia Studio.
    - **MIDI Export:** High-quality `.mid` files for any DAW (Ableton, FL Studio, Logic, etc.).
- **Dynamic BPM Control:** Adjust the tempo of your generated beat in real-time.

## 🚀 Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/z9mb83/beat-pro.git
   ```
2. **Open the App:**
   Simply open `index.html` in any modern web browser.

## 🛠️ Built With

- **HTML5 & Vanilla CSS3** (Custom design system with glassmorphism)
- **JavaScript (ES6+)**
- **MIDI.js / Custom Export Logic**

## 📂 Project Structure

- `index.html`: Main application interface.
- `style.css`: Modern, responsive design system.
- `app.js`: Core logic for lyric analysis and sequencer.
- `lmms-export.js`: Logic for generating LMMS project files.
- `midi-export.js`: Logic for generating MIDI files.
